# Lab 04

## Temat
Maszyna Stanów i Kompletna Pętla

## Autor
Aliaksei Makarevich

## Opis
Laboratorium 04 wykonane w środowisku microStudio z użyciem języka microScript.

Zaimplementowano:
- maszynę stanów gry opartą o delegaty funkcji (`current_update`, `current_draw`),
- ekran menu głównego,
- ekran rozgrywki,
- ekran zwycięstwa,
- ekran porażki,
- reset gry i przełączanie stanów,
- interfejs HUD z punktacją, liczbą żyć i hi-score,
- trwały zapis najlepszego wyniku z użyciem `storage`.
