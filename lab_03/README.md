# Lab 03

## Temat
Dynamika, kolizje i Frontline

## Autor
Aliaksei Makarevich

## Opis
Laboratorium 03 wykonane w środowisku microStudio z użyciem języka microScript.

Zaimplementowano:
- pociski gracza,
- detekcję kolizji AABB,
- mechanizm garbage collection metodą Mark & Sweep,
- pociski przeciwników,
- logikę Frontline,
- stan końca gry i restart,
- zadania dodatkowe: bunkry oraz efekt cząsteczek.
